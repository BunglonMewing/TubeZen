Kebijakan Privasi

PipeTube tidak mengumpulkan data pribadi. Semua permintaan video dikirim langsung dari perangkat ke YouTube tanpa server perantara.

Izin Aplikasi

  • Penyimpanan: menyimpan file video/audio.
  • Jaringan: mengunduh konten YouTube.

Tidak ada analitik pihak ketiga atau pelacakan iklan.
